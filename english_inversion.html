Tipo 1 — Adverbios negativos

Con Never, Rarely, Seldom, Hardly, Scarcely, Not only, Little, No sooner al inicio, el sujeto y el auxiliar se invierten.

Adv. negativo + AUXILIAR + SUJETO + verbo
I have never seen this. ↓ invertida Never have I seen this.
She rarely goes out. ↓ invertida Rarely does she go out.
Tipo 2 — Condicionales invertidas

Se elimina el if y se invierte el auxiliar. Tres formas:

Had + sujeto + participio… (if had)
Were + sujeto + to/adj… (if were)
Should + sujeto + infinitivo…
If I had known, I would have helped. ↓ invertida Had I known, I would have helped.
If you should need help, call me. ↓ invertida Should you need help, call me.
Tipo 3 — So / Neither / Nor

Para mostrar acuerdo con lo que dice otra persona.

So + AUXILIAR + sujeto → acuerdo positivo
Neither/Nor + AUXILIAR + sujeto → acuerdo negativo
I love pizza. — Yo también. ↓ So do I.
I can't swim. — Yo tampoco. ↓ Neither can I.
Tipo 4 — Lugar / Dirección

Con Here, There, Down, Up, Out, In al inicio, el verbo va antes del sujeto.

Adv. de lugar + VERBO + SUJETO nominal
The bus comes here. ↓ Here comes the bus!
